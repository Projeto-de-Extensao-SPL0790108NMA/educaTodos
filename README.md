# EducaTodos

EducaTodos é um projeto que visa proporcionar uma plataforma de aprendizado colaborativo, focando na reestruturação de presos em cadeias.

## Estrutura do Projeto

- **backend/**: Contém a lógica do servidor e a base de dados.
- **frontend/**: Contém a interface do usuário e a lógica do cliente.

## Como Começar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Projeto-de-Extensao-SPL0790108NMA/educaTodos.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd educaTodos
   ```
3. Instale as dependências do backend:
   ```bash
   pip install -r backend/requirements.txt
   ```
4. Inicie o servidor:
   ```bash
   python backend/manage.py runserver
   ```
5. Para o frontend, instale as dependências:
   ```bash
   cd frontend
   npm install
   ```
6. Inicie o aplicativo:
   ```bash
   npm run dev
   ```
